O modulo de Textos permite criar documentos de texto formatado para suas orientacoes, receitas, laudos e mensagens. Voce escreve uma vez, salva como template e reutiliza quando precisar.

## Visao geral

Textos sao conteudos escritos pelo nutricionista que podem ser exportados em PDF ou HTML, copiados para a area de transferencia ou impressos. Cada texto pertence a um dos subtipos abaixo, e voce pode criar quantos quiser.

## Subtipos de texto

O DietFlow organiza seus textos em quatro categorias:

| Subtipo | Para que serve | Editor |
|---------|---------------|--------|
| **Mensagens** | Textos para enviar ao paciente (WhatsApp, email) | Editor de mensagens com variaveis |
| **Observacoes** | Anotacoes clinicas e observacoes internas | Editor de texto rico |
| **Documentos** | Laudos, relatorios e documentos formais | Editor de texto rico |
| **Gerais** | Qualquer outro tipo de texto | Editor de texto rico |

## Editor de texto rico

Para os subtipos Observacoes, Documentos e Gerais, o DietFlow oferece um editor completo com:

- **Formatacao** — negrito, italico, sublinhado, tachado
- **Listas** — com marcadores ou numeradas
- **Titulos** — hierarquia de cabecalhos
- **Alinhamento** — esquerda, centro, direita
- **Sobrescrito e subscrito** — para formulas e referencias
- **Historico** — desfazer e refazer alteracoes

A barra de ferramentas fica fixa no topo do editor. Ao selecionar um trecho de texto, um menu flutuante aparece com opcoes rapidas de formatacao.

## Editor de mensagens

O subtipo **Mensagens** usa um editor especial, otimizado para comunicacao com pacientes. Voce escreve o texto e insere **variaveis dinamicas** que serao substituidas automaticamente:

- **Nome do paciente** — insere o nome completo
- **Data da consulta** — insere a data formatada
- **Profissional** — insere seu nome

As variaveis aparecem como chips clicaveis. Um preview ao vivo mostra como a mensagem ficara com os dados reais.

<Callout type="info" title="Limite de caracteres">
Mensagens destinadas ao WhatsApp respeitam o limite de 4096 caracteres. O contador aparece em tempo real no editor.
</Callout>

## Importar templates

Ao criar um novo texto, voce pode importar um template existente como ponto de partida. Clique em **Importar** no topo do editor, escolha o template desejado e o conteudo sera carregado automaticamente.

## Salvar e organizar

Cada texto e salvo automaticamente ao fechar o editor (se houver alteracoes). Voce pode:

- Editar o **titulo** clicando nele no topo
- Adicionar **tags** para facilitar a busca
- Acessar todos os seus textos em **Meus Conteudos → Favoritos → Textos**

## Exportacao

Textos (exceto mensagens) podem ser exportados em diversos formatos:

- **PDF** — para imprimir ou enviar como arquivo
- **HTML** — para visualizar no navegador
- **Copiar texto** — copia o conteudo formatado para a area de transferencia
- **Imprimir** — envia direto para a impressora

Para exportar, clique no botao de exportacao na barra de acoes do editor.

<Callout type="info" title="Mensagens nao exportam">
O subtipo Mensagens nao tem opcao de exportacao porque e feito para ser copiado e enviado diretamente ao paciente via WhatsApp ou email.
</Callout>
