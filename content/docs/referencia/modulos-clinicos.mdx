O DietFlow organiza o trabalho clinico em modulos independentes que se comunicam automaticamente. Cada modulo cobre uma etapa do atendimento nutricional — da anamnese ao plano alimentar.

Os tres modulos centrais formam o [Pipeline Clinico](/docs/referencia/pipeline-clinico): Antropometria alimenta o Calculo, que alimenta a Dieta. Os demais modulos funcionam de forma independente, mas todos compartilham o mesmo sistema de [favoritos e templates](/docs/referencia/favoritos-templates) e [exportacao](/docs/referencia/exportacao).

## Como os modulos se conectam

Os modulos do pipeline se alimentam automaticamente — voce registra as medidas, o sistema calcula e distribui:

```
Antropometria → Calculo Nutricional → Plano Alimentar
```

Quando o peso do paciente muda, basta criar um novo calculo importando os dados atualizados. O sistema recalcula TMB, GET e VET automaticamente, e as novas metas podem ser importadas no plano alimentar. Veja o fluxo completo em [Pipeline Clinico](/docs/referencia/pipeline-clinico).

Todos os modulos que geram registros clinicos suportam [exportacao](/docs/referencia/exportacao) em HTML interativo, PDF e texto. E todos (exceto Calculo) permitem salvar como [favorito](/docs/referencia/favoritos-templates) para reutilizar entre pacientes.
