Atalhos disponiveis para agilizar a navegacao pelo DietFlow.

## Navegacao geral

| Atalho | Acao |
| --- | --- |
| `Ctrl+K` (ou `Cmd+K` no Mac) | Busca global |
| `Tab` | Navegar entre elementos |
| `Enter` | Confirmar selecao |
| `Esc` | Fechar modal ou cancelar acao |

<Callout type="info">
Esta pagina sera expandida conforme novos atalhos forem implementados no sistema.
</Callout>
